# Habbit Tracker App

# Description

A Habbit Tracker app that allows to add and remove habbits  and on the go also allows to track the previous 7 days status of a sspecific habbit.

# Features

- Add Habbit.
- Remove Habbit 
- Track habbit status 
- Allow user to change current and previous days status to Done,Not Done and No Action

# Tech Used

HTML,CSS,Javascript,jQuery
NodeJS,MongoDB,Express ans EJS 

# How to Install

- Clone the project onto your local machine.
- Run 'npm install' to install required dependencies.
- Run 'npm start' in terminal to start server.
- Visit your app at http://localhost:8000.

# Action Routes

- POST : /add >> Add Habbit
- GET : /delete/:id >> Delete Habbit
- GET : /update >> Update Habbit
- GET : /weekly >> Entering Weekly View
